This repository is configured with **GitHub Actions** for automatic deployment.

1.  Push this code to a GitHub repository (e.g., `username.github.io`).
2.  Go to **Settings** > **Pages**.
3.  Under **Build and deployment**, select **GitHub Actions** as the source.
4.  The action will run automatically on push to `main` and deploy your site.
